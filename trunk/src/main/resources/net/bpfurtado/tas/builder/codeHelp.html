Scene Code Programming

It is possible to add much more interactivity to your Text Adventure by adding Beanshell scripts to be executed when openning a new scene.

To create 'Scene Actions'

The tab named 'Actions' has code that is executed everytime the scene is openned. The tab name 'Assertions' has code that is executed every time some Scene is openned.



The following code would go well on some Scene Action code:
  • Set main attributes: 
            player.setStamina(player.getStamina()+6);

    
        player.setSkill(player.getSkill()-1);

    
        player.setStamina(20);

    
        player.setSkill(6);
  • Update main attributes (dec & inc operations are available for Stamina, Luck, Skill and Damage): 
            player.incStamina(1);

    
        player.decStamina(1);

    
        player.incSkill(2);

    
        player.decSkill(2);

    
        player.incLuck(3);

    
        player.decLuck(3);

    
        player.incDamage(4);

    
        player.decDamage(4);
  • Add 2 attributes: 
            player.addAttribute("long-sword");

    
        player.addAttribute("apples", 3);
  • Add and Remove attributes: 
            player.addAttribute("iron shield");

    
        player.removeAttribute("horse");
  • Test if the player has some attribute: 
            if(player.has("retractable claws")) {

    
            pathsToHide.add(3); //Hide the 'climb the dungeon wall' path.

    
        }
  • Check some attribute value and Hide a scene path: 
            if(player.getIntValue("apple")==1) {

    
            pathsToHide.add(2);

    
        }
  • Increase some attribute int value: 
            ammo = player.getIntValue("gun ammo");

    
        player.addAttribute("gun ammo", ammo + 6);
    Or 
            player.incIntValue("coins", 30);
  • Decrease some attribute int value: 
            n = player.getIntValue("health potion");

    
        player.addAttribute("health potion", n-1);
    Or 
            player.decIntValue("stamina", 6);
  • Change the scene text based on some attribute value: 
            if ( player.getIntValue("cat statues")==2 ) {

    
            text="yet 2 cat statues?";

    
        } else {

    
            text = "no 2 anymore... but " +  player.getIntValue("cat statues");

    
        }
